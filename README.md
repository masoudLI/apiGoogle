# Le projet

FreeReads est un **api** créé avec **Symfony 7** qui permets de récuperer des 
livres depuis **google Books**

## Technologies

FreeReads sera développé en utilisant :

- Docker
- Symfony 7
- Mysql 8.0
- PHP 8.2
- PHPMyAdmin
- Sass
